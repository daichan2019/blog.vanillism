## ブログを作った

こんにちは、@vanilla.dev と申します。
業務ではフロントエンド、バックエンド問わず色々触らせてもらってますが、
React と Next.js の素振り環境がほしいなと日々感じていたので、簡易的なブログを作ってみました。

Next.js と Tailwind.css をベースに、記事管理は MicroCMS を使ってみることにしました。
API の型定義からリクエストメソッドを定義できる Aspida を使ってみたかったのが MicroCMS を採用した主な動機ですが、単純な記事管理だったらマークダウンのほうが便利だったかも、なんて思ってます。

Next.js は ver13 になって大幅にアップデートが入り、色々試したいなと思ってブログを作ってみたものの、リリース急ごうと思って全然試せてないです。。。

気軽に壊せる環境がとにかく欲しかったのでこのブログで色々試して見たい所存。。。
